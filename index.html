Proyectos

Wordle

En este proyecto desarrollé un Wordle, que funciona realmente como la versión online, pero yo lo desarrollé con python, los inputs y outputs se muestran en consola.

Apuntado

El 'Apuntado' es muy popular en mi ciudad natal (Medellín), es una versión con algunas modificaciones del juego 'Terna, terna, cuarta y sobrante'. Lo desarrollé con Python, usando Tkinter para la interfáz de usuario.

Landing page Cultivo de aguacate Hass

Esta es una página web responsive, desarrollada con HTML y CSS, en la que se muestra la información más relevante de una empresa exportadora de aguacate Hass.

Galleta de la fortuna

Esta página de la galleta de la fortuna, desarrollada con HTML, CSS, JavaScript y React, después de un click del usuario, cambia la interfaz y muestra una frase aleatoria.

El Clima en tu ubicación

El clima en tu ubicación, es una página responsive desarrollada con HTML, CSS, JavaScript y React y por medio de la API Geolocation de los navegadores, y la API Open Weather Map, se le indica al usuario cuál es el clima en la zona que está actualmente.

Personajes y Mapas Rick & Morty

La página de Personajes y Mapas Rick & Morty, es una página web responsive, desarrollada con HTML, CSS, JavaScript y React, y por medio de la API de Rick y Morty, se despliegan todos los personajes de alguno de los 126 mapas del programa, después de un input del usuario.

CRUD de usuarios

El CRUD de usuarios, es una página web responsive desarrollada con HTML, CSS, JavaScript, React y React-Hook-Form, como sus siglas lo indican, permite crear, leer, actualizar, y borrar usuarios. Esta fue desarrollada con una API, cuya documentación se encuentra aquí.

Pokedex Page

La Pokedex Page, es una página web responsive, desarrollada con HTML, CSS, JavaScript, React, React-Router-Dom y Redux Toolkit, en la que después de que el usuario ingresa su nombre, puede acceder a todos los pokemones, los puede filtrar por tipo y por cadenas de caracteres en su nombre, y puede ver las características y estadísticas a detalle de cada uno.
